# Student Grading System

A C++ program to store student data, calculate grades, and display a report.

## Features
- Store multiple students' data  
- Calculate grade automatically:  
  - **A**: 90+  
  - **B**: 75–89  
  - **C**: 60–74  
  - **D**: 40–59  
  - **F**: Below 40  
- Display all students in a neat table format  

## How to Run
```bash
g++ main.cpp -o grades
./grades
